Kron-Face

an intuitive open-source user interface (UI) built for cron to exemplify the experiance of scheduling jobs in Linux, Windows and Mac enviornment.It can be useful for administrators and developers as an add on for conviniently scheduling jobs and keeping tracks of it.

Intuitive UI

New asthetics of scheduling jobs for a globally integrated enviornment.

Opensource

For the community, By the community. Totally free and open to suggestions, customizations and upgrades.

Experiance

Changes how people schedule jobs using Cron from unix like scripting to easy, effective, globally accessable web application.

Cron

It runs on top of Crontab (an opensource unix-like job schedular), so Cron should be installed on the server to schedule jobs.

Apache

The app can be deployed using any web contained like tomcat, which is lightwieght and easy to handle.

Convinient

Just provide details for when you want the job to run and a link to the runnable file and its done. The app will auto schedule the jobs for you without.
